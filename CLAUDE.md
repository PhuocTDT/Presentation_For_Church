## Project Overview

EasyWorship (easyworship-app) — an Electron desktop application for church worship presentation. Manages songs, schedules, Bible verses (Vietnamese), and background media for live projection.

## Commands

```bash
# Install dependencies
npm install

# Run the app
npm start          # launches Electron
```

There is no build step, linter, or automated test framework configured. The app runs directly via Electron with no bundler.

## Architecture

**Electron app with two processes:**

- **Main process** (`main.js`): Window creation, IPC handlers, file I/O, custom `app-media://` protocol for serving local media files. Data stored in Electron's `userData` path (`songs.json`, `settings.json`, `media/` folder).
- **Renderer process** (`index.html`): All UI logic lives in a single HTML file (~1370 lines) with inline `<script>` and `<style>` blocks. No framework — vanilla JS with DOM manipulation.

**UI Layout** (3-panel + bottom row):
- Left: Schedule list (worship order)
- Middle: Preview panel (slide thumbnails)
- Right: Live output panel
- Bottom: Song library tab / Bible tab / Media library

**Key data flow:**
1. Songs loaded from `songs.json` via IPC → displayed in library
2. Songs added to schedule → slides generated by splitting lyrics on `\n\n`
3. Slides selected in preview → sent to live output via `goLive()`
4. Background media applied via `setBackground()` using `app-media://` protocol URLs

**Secondary file:** `edit-song.html` — modal song editor with contenteditable slide blocks and formatting toolbar.

## Data Formats

- **Songs**: `{ id, title, lyrics }` — lyrics use `\n\n` as slide delimiter
- **Schedule files** (`.bcsch`): JSON array of `{ id, title, lyrics, scheduleId }`
- **Bible**: `data/Bible_Vietnamese.xml` parsed client-side via DOMParser

## Key Implementation Details

- Node integration enabled, context isolation disabled — renderer has full Node.js access
- Tailwind CSS loaded via CDN (not built locally)
- Font sizing uses custom `<span>` wrapper instead of execCommand's 1-7 scale
- Resizable panels implemented with mouse event tracking on `.resizer` elements
- Media import copies files to userData media folder; served via custom protocol
- Vietnamese Bible with book name mapping for display
